	
	QuickRoutine 1.2
	----------------

Please be sure that you have JAVA run time environment installed in 
you computer before execute any *.jar file (e.g. QuickRoutine 1.2.jar),
Otherwise install it before running QuickRoutine. JAVA Run time's backup 
should be available in your QuickRoutine's CD.

#######################################################################

	Step1: Install >> jre1.5.exe from the CD.     
	Step2: Run >> QuickRoutine1.2.jar
	
#######################################################################

This CD provide example routine 2004 of Dept. of Computer Science
and Engineering (CSE), Shahjalal University of Science & Technology.

To open these routines follow the steps below :
      # To open example routine (Without manual routine) access the 
folder “Example Routine Data” and open “example_Routine.rtn” file.
      # To open example routine (With manual routine) access the 
folder “Example Routine Final” and open “example_Routine.rtn” file.
