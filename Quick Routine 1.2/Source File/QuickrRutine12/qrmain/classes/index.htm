What is QuickRoutine all about?

QuickRoutine is very user-friendly software to help you making a class routine for all the batches' of a semester. Providing all necessary information it will give you a complete class routine automatically of all the current batches, teachers and rooms. If you have any experience of making class routine by hand then you must know that necessary information to make a semester's full class routine is available rooms, existing batches, available teachers and all courses information of current semester. To give you full control of generating routine you can make part or full of the routine manually with the help of QuickRoutine and let QuickRoutine generate the rest automatically. Note that QuickRoutine doesn't generate lab course's routine automatically. So you need to assign all lab courses' classes manually.

How to start?

To create a new routine click File on the menu bar and click New. During create a new routine - enter the number of semesters per year of your university. Enter maximum sections of a semester. Then you have to provide all rooms, batches, teachers and course information. After providing this basic information you can make part or full of your class routine manually with the help of full graphical interface of QuickRoutine. As we have found that lab course's routine requires a lot more information available to know the user intention of lab class routine and this varies from department to department, we have leaved this job to user to manually assign those classes. You can also assign classes manually for theory classes. After inserting all of this information click on Generate Automatic Routine button of the main editor. Now click on the left sides' tree of the main editor to view the routine. You can edit class routine now. View the report always. You can delete/update information.

How to insert rooms?

Click Insert on menu bar and then click Room Information. You need to provide a room ID. A sample room ID is '309A'. Room ID should be small for keeping your routine looks slim. Room description is must for lab room cause QuickRoutine 1.1 does not assign class automatically to lab rooms. So you always need to assign class to lab room manually. And you will find it helpful to have a brief description of room during assigning a class. A sample description is 'Network lab'. If you don't have a room available for fulltime you can let it know QuickRoutine by inserting session. To know more about session click here. Please note that QuickRoutine 1.1 assign theory classes to theory room. Neither lab courses nor lab rooms are handled automatically by QuickRoutine 1.1. If you need any theory courses' class need to assign to any lab class please do it manually.

How to insert batches?

Click Insert on menu bar and then click Batch Information. Insert all batches of current semester. You may find it new to insert the "class start at" attribute of a batch. It is a routine style of a batch. Only two routine style is available for version 1.1. If you have more idea please write to us at tojewel@hotmail.com with full explanation. If you don't find required semester or section, remember that you enter maximum semester and maximum section during create a new routine. You can also bias a batch's routine style by inserting session of a batch. For example if batch: 1/1-B doesn't want to attain any classes at 1:00 pm, set those sessions Busy. QuickRoutine will take this account during making a automatic routine and let it know during assign a class manually of that batch.

How to insert teachers?

Click Insert on menu bar and then click Teacher Information. A teachers' ID is must. Teacher ID can be formed with first letters of his/her names' word. For example Muhammad Zafar Iqbal can be identified with a ID: MZI. If you have a part time teacher and want QuickRoutine to assign class for that teacher at allowed time, inert this information by session panel. For example you have a teacher only for Sunday and Monday vacant, set Sunday and Monday Free and all other day Busy. Please be clear that what is session for a teacher.

How to insert courses?

Click Insert on menu bar and then click Course Information. All course ID must have to be distinct. If you have same course ID for theory and lab then you can add a 'L' after the lab course ID. For example CSE-100 is a theory course ID and CSE-100L is corresponding lab course ID. Same course ID for different section can be identified as CSE-123A, CSE-123B and so on. Before inserting a course you must have corresponding batch information available. If you keep a course teacher 'Assign Later', QuickRoutine will not generate automatic classes for this course. Please don't think too much about course room. Just let it be 'Assign Later' for now. We think all theory class's duration should be 1 hour long. If you have an exception please insert that course as a lab-course.

What is session in input panel and how it can be helpful?

Session is a full routine's layout. By using this field you can let QuickRoutine know that at when a room or batch or teacher is free. For example you want a newly married teacher's class should be started always at 10 am. So insert that teacher's data setting all 8 and 9 m's periods busy by only clinking corresponding periods on the session panel. Click on the periods and day title to set the values quickly. Be careful that setting too much Busy sessions for a batch or a teacher or a room may cause QuickRoutine unable to generate complete routine without violating some constraints. You will also aware of this busy session during assigning class manually or editing automatically generated routine.

How to assign classes manually?

Click Manual on menu bar and then click Batch Routine to assign class batch wise or Teacher Routine to assign class teacher wise. Now select a batch or teacher. Corresponding courses will be shown. Select a course, select a room and select the periods from routine layout. Just follow the tips of the panel below to know what to do when during assign class manually. To remove a class - select that class first and then click 'Remove' button. As you have to assign all lab course manually, to check the status of manual routine click on the report button. To move a class to a new suitable place - select that class and drag and drop that class to a new completely free period.

What does it means "Editing Class Routine"?

Editing a class routine means you can select a class from routine and drag and drop that class to a new suitable place. After selecting a class all other same courses class will be highlighted and selected course's batch, room and teacher status will be shown to let you know whether you can move that class. Status means - whether corresponding batch or teacher or room is busy or assigned to another class. So that you can easily decide at which you can transfer that class If you find that you can't move a class only for the current assigned room is busy or assigned to other class at your desired place and so you want to change the room, just assign that class manually. During editing don't forge to check the report always.

What is a routine report?

As we experienced that making a class routine is just handling a big amount of data. So you always want to know that whether all of the classes of all of the courses have assigned perfectly. A report can provide you this information. As you have to assign all lab courses manually so check always the lab routine report to know if there is any lab class left to assign? Check also automatic routine report always cause though QuickRoutine must pick a routine under some constraints but it may fail to assign classes to some of the theory classes if there is lack of class rooms' or teachers' free session or if it violate any of the constraints of automatic routine generating algorithm.

What are the constraints of automatic routine generating algorithm?

During generating automatic routine QuickRoutine must generate the best possible routine without violating some constraints. For example QuickRoutine will not assign more than one class of a course on a same day. To assign a class on a day it requires a period at which the batch and the teacher is free and a free theory room is available at that periods. Of course you can assign more than one classes of a course manually. So that it is very important to check the both automatic and manual routine report always. Because automatic routine report will let you know that how much classes is not possible to assign for not violating this constraints of a routine.

What does it means "Class Start at Sunday Morning"?

It means that a batch's class will start on Sunday at morning and continue until lunch. On Monday day that batch's class will be started after lunch and continue until evening and on Tuesday that batch's class will be started at morning again and so on. If you enter the class style "Sunday Evening" for a batch that batch's class will be started on Sunday at evening and on Monday at morning and so on. If you follow this class style it will give students a consecutive class everyday, so that a regular student can save the wasted time between two classes and a student who have a drop course can participate his/her drop classes. Of course this is just setting a priority. QuickRoutine will try best to maintain this style. This excellent idea is given to us by Dr. Kabir Hossain, Assistant Professor, Dept. of Statistics, Shahjalal University of Science and Technology.
